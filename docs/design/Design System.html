Foundations

A quiet, evidence-led
system for finding warm local prospects.

Two surfaces, one voice. An internal operator app built for scanning hundreds of rows quickly, and a public report that reads like a senior consultant's audit — never a SaaS pitch.

Evidence over enthusiasm

Show the operator (and the prospect) the data. Numbers are tabular, sources are cited, screenshots are attached. No exclamation marks.

High score = warm lead

Our scores measure weakness. The colour scale gets warmer as the opportunity gets larger — never "green is good".

Calm in dense places

Tables are roomy and scannable. Colour is reserved for signal: severity, status, and the warm-lead accent. Everything else is paper and ink.

01 — Colour

Paper, ink & a single warm signal.

A monochrome palette tinted slightly warm (h ≈ 60–80) so the page never feels clinical. One ochre accent does the heavy lifting for "warm lead", supplemented by severity tones on the public report.

02 — Typography

An editorial display serif & a workmanlike sans.

Newsreader carries every headline and the public report's authoritative voice. IBM Plex Sans handles dense interface text. IBM Plex Mono is reserved for tabular figures, identifiers, and timestamp/eyebrow labels — never paragraphs.

08 — Annotation layer

The pin, the callout & the legend.

Numbered pins float above the prototype when annotation mode is on. Each pin opens a dark callout describing what the operator should learn — what the score means, which action is primary, what error states to expect.

Combined score = warm-lead probability

Higher is "worse for the business, better for you." 71+ is automatically tinted with the accent — these are the rows operators should action first.

2
GBP & A11y sub-scores

Same scale, never inverted. If one sub-score is > 70 and the other < 40, the dominant angle pill labels the pitch direction (GBP or Accessibility, never "Both").

3
Warm-lead row tint

Soft accent wash from the left edge appears once a prospect's public report has been opened. It overrides the standard hover state.

4
Audit-failed state

Dimmed row, red sub-meta, single "Retry" action. The prospect is never auto-deleted — operators decide if a 503 is a permanent skip.
